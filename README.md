## Installation

Install the program using the following command:
```bash
poetry install
```

## Usage

You can start the program using:
```bash
poetry run invoke start
```

or run the tests:
```bash
poetry run invoke test
```

or generate a coverage report:
```bash
poetry run invoke coverage-report
```
